nuget Package Managaer
====

c#開発においてnugetパッケージ作成を支援します。

## Description
ローカルリポジトリを検索して以下の操作を行います。
*.nuspecファイルが存在しないc#プロジェクトがあれば、.nuspecファイルのテンプレートを自動生成します。
*バイナリが更新されているリポジトリがあればパッケージを作成します。

## Requirement
*.Net Framework 4.6

## Usage
NugetPackager.exeを実行します。
